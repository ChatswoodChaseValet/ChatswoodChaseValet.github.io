🔄 Auto-Save to File

Save your records to a file on this PC automatically — every time you log a car, edit, or check out a vehicle. Recommended: save the backup file inside your OneDrive, Google Drive, or Dropbox folder so it also gets cloud-backed.

○ Auto-save off — using browser storage only
How it works:
  • Chrome / Edge: Silent file save to a folder of your choice (recommended)
  • Other browsers: Auto-download to Downloads folder (look for the latest chatswood-valet-backup.json)
  • If you restart the browser, click Re-enable once to restore permission
  • Auto-save runs in addition to browser storage — you have two copies for safety
