🖨 Key-Tag Printer (BIXOLON)

Print the key tag (ticket #, bay, plate + scannable barcode) straight to a BIXOLON SPP-R200III. Because this is a web app, a small free bridge app must run on this device:

One-time setup on this device:
  1. Android: install “mPrintServer” from the Play Store. Windows: install the Web Print SDK utility from BIXOLON’s SPP-R200III download page.
  2. Pair the SPP-R200III to this device over Bluetooth, then add it in mPrintServer and give it a name. In its Web Server settings make sure the server is ON (port 18080).
  3. Type that exact name below, Save, then tap Test Print.
Leave Connection on HTTP and address as 127.0.0.1:18080 to start. If printing is blocked because this site is HTTPS, switch the WebPrint app to its secure mode and pick HTTPS here (adjust the port if it differs).
